- 黑名单过滤
- API调用和分析
- 数据存储

### P1（高优先级）
- 智能去重
- Web统计面板
- 数据清理
- 错误处理

### P2（中等优先级）
- 可视化图表
- 配置向导
- 日志系统
- 性能优化

### P3（低优先级）
- OCR内容过滤

## 风险与应对

| 风险 | 应对措施 |
|------|----------|
| API成本超预算 | 实现严格缓存和去重，测试阶段使用mock |
| 窗口监听不稳定 | 多重检测机制，降级为纯定时模式 |
| 数据库性能问题 | 定期清理，索引优化 |
| 打包体积过大 | 精简依赖，使用PyInstaller优化选项 |
